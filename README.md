# Guia de Redes de Computadores
Projeto acadêmico desenvolvido com o objetivo de disponibilizar um ambiente interativo para estudo dos principais conceitos e protocolos de redes de computadores.

## Sobre o Projeto
O intuito deste projeto é disponibilizar um ambiente que promova o estudo dos alunos na disciplina de Redes de Computadores, reunindo os conteúdos utilizados em apresentações realizadas em sala de aula.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- Tailwind CSS
- JavaScript (básico para interações)
